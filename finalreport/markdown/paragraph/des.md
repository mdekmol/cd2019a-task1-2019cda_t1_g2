V-rep 動態模擬
===
送球機構設計與模擬

手足球發球與進球後自動送球機構設計與 V-rep 動態模擬 (機構與傳動系統設計與模擬)

可採用螺旋機構或多連桿機構設計, 透過進球感測器感應後, 將儲存於螺桿上的球送回球檯.

以下為參考機構模擬影片:

系統功能展示



改寫code=python to lua並簡化

優點:

python : 可執行運算值較大的編譯

lua : Vrep內部沿用 , 延遲現象較少

缺點:

python : 會有爆ping問題導致延遲

lua : 太多運算時直接停止
